Проанализируй следующие сообщения из рабочего группового чата "{chat_title}" и найди ВСЕ поручения (задачи, указания, просьбы что-то сделать).

Поручение — это когда один человек просит/требует/поручает другому выполнить конкретное действие.

Примеры поручений:
— «Иванов, подготовь отчёт к пятнице»
— «Нужно связаться с клиентом и уточнить»
— «Проверь, пожалуйста, документы по объекту»
— «Завтра до обеда отправь заявку»

НЕ является поручением:
— Информирование: «Встреча перенесена на 15:00»
— Обсуждение: «Как думаете, стоит ли...»
— Отчёт: «Я сегодня закрыл 5 заявок»

Для КАЖДОГО найденного поручения верни JSON-объект:
- source_message_id: int — ID сообщения-источника
- assigner_user_id: int — кто поручает
- assignee_user_id: int | null — кому поручают (null если не ясно)
- task_text: string — краткая формулировка поручения (до 200 символов)
- confidence: float — уверенность от 0.0 до 1.0
- due_date: string | null — дедлайн в формате YYYY-MM-DD (null если не указан)

Верни JSON-массив. Если поручений нет — верни пустой массив [].

ВАЖНО: Формат ответа — ТОЛЬКО валидный JSON-массив, без пояснений.
