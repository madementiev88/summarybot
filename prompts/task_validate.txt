Проверь статусы открытых поручений на основе переписки в чате "{chat_title}".

Для каждого поручения определи:
- Выполнено ли оно (есть подтверждение в переписке)?
- Просрочено ли оно (дедлайн прошёл)?
- Всё ещё актуально?

Открытые поручения:
{tasks_json}

Переписка за последние 24 часа:
{messages_text}

Сегодняшняя дата: {today}

Для КАЖДОГО поручения верни JSON-объект:
- task_id: int — ID поручения
- new_status: "open" | "closed" | "overdue" — новый статус
- close_message_id: int | null — ID сообщения, подтверждающего выполнение (null если нет)
- reason: string — краткое пояснение решения (до 100 символов)

Верни JSON-массив. Если ничего не изменилось — верни пустой массив [].

ВАЖНО: Формат ответа — ТОЛЬКО валидный JSON-массив, без пояснений.
