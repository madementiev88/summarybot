Сгенерируй подробный текстовый контент для каждого слайда презентации.

План презентации:
{plan_json}

Для каждого слайда верни JSON-массив. Ответ — ТОЛЬКО валидный JSON, без markdown-обёрток:
[
  {{
    "slide_number": 1,
    "title": "Заголовок",
    "body_text": "Основной текст (1-2 предложения, если нужен)",
    "bullet_points": ["пункт 1", "пункт 2", "пункт 3"],
    "speaker_notes": "Заметки для докладчика"
  }}
]

Правила:
- Деловой русский язык
- Каждый слайд: 3-6 пунктов или 2-3 коротких абзаца
- body_text — опционально, для слайдов где нужен сплошной текст
- bullet_points — основной формат подачи информации
- speaker_notes — расширенные пояснения для докладчика
- Не повторяй заголовок в тексте слайда
