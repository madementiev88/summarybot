Сгенерируй подробный текстовый контент для каждого слайда презентации.
Презентация в корпоративном стиле Сбера (зелёный акцент, строгий деловой стиль).

План презентации:
{plan_json}

Для каждого слайда верни JSON-массив. Ответ — ТОЛЬКО валидный JSON, без markdown-обёрток:
[
  {{
    "slide_number": 1,
    "title": "Заголовок",
    "body_text": "Основной текст (1-2 предложения, если нужен, иначе пустая строка)",
    "bullet_points": ["пункт 1", "пункт 2", "пункт 3"],
    "metrics": [
      {{"value": "+25%", "label": "рост продаж"}},
      {{"value": "×2.5", "label": "эффективность"}}
    ],
    "badge": "+15%",
    "speaker_notes": "Заметки для докладчика"
  }}
]

Правила:
- Деловой русский язык, стиль Сбера
- Каждый слайд: 3-6 пунктов или 2-3 коротких абзаца
- body_text — опционально, пустая строка если не нужен
- bullet_points — основной формат подачи информации
- metrics — массив крупных числовых показателей (value + label), используй только там где уместны цифры/KPI. Если нет метрик — пустой массив []
- badge — короткая динамика для бейджа в правом верхнем углу слайда ("+12%", "×3", "ТОП-1"). Если не уместно — пустая строка ""
- speaker_notes — расширенные пояснения для докладчика
- Не повторяй заголовок в тексте слайда
- Числа и метрики округляй до целых или одного знака
